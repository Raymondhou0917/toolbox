# 🧰 雷蒙三十工具箱（Toolbox）

> 超級個體的實用小工具——簡單、美觀、好用。

這是一個我為自己和社群夥伴打造的數位工具集合。裡面放的都是我們在工作、教學、活動中會用到的小工具，目標是讓這些工具**免費、好用、隨開即用**。

## 🎯 目前有哪些工具？

| 工具 | 說明 | 路徑 |
|------|------|------|
| **抽獎工具集** | 包含幸運輪盤、數字抽獎、配對分組三種功能。支援自訂名單組、登入後可雲端同步。 | `/lottery` |
| **倒數計時器** | 簡潔的課程倒數計時器，有休息、開場、練習三種模式，可以嵌入 Notion 或 OBS。 | `/timer` |

未來會持續新增更多工具，像是：簡報計時器、隨機點名器、活動簽到表等等。

## 🛠️ 技術架構（給想改 code 的人看）

這個專案用的是：
- **前端**：React 19 + Tailwind CSS 4 + shadcn/ui
- **後端**：Node.js + Express + tRPC
- **資料庫**：MySQL（透過 Drizzle ORM）
- **登入**：Google OAuth

專案結構很單純：
- `/` → 入口頁（工具選單）
- `/lottery` → 抽獎工具集（React SPA）
- `/timer` → 倒數計時器（原本的純 HTML/CSS/JS，直接放在 public 資料夾）

## 🚀 怎麼部署？

我是用 [Zeabur](https://zeabur.com/) 部署的，步驟大概是：
1. 把這個 repo 連結到 Zeabur
2. 設定環境變數（資料庫連線、OAuth 等）
3. 綁定自訂網域（例如 `tool.lifehacker.tw`）

如果你想自己跑起來玩玩，可以：
```bash
pnpm install
pnpm dev
```

## 📝 Update Log（更新紀錄）

### 2024-12-09｜v1.0.0 首次發布

**這次做了什麼？**

終於把工具箱整合起來了！原本抽獎工具和倒數計時器是分開的兩個專案，現在合併成一個，方便管理和部署。

**主要功能：**
- ✅ 入口頁面（Bento Grid 風格，可以選擇要用哪個工具）
- ✅ 抽獎工具集（輪盤、數字、分組三合一）
- ✅ 名單組功能（可以儲存常用名單，像是「雷蒙三十成員」、「健身鍛鍊項目」）
- ✅ Google 登入 + 雲端同步（登入後名單會存到雲端，換電腦也能用）
- ✅ 本地暫存（沒登入也會自動存在瀏覽器，重新整理不會消失）
- ✅ 倒數計時器（保留原本的設計，沒有重構，直接搬過來）

**為什麼這樣做？**

原本想用 Monorepo 把所有工具統一用 React 重寫，但發現倒數計時器重構後跑版嚴重（按鈕大小不一、排版亂掉）。所以改用「方案 B」：把原本的 HTML/CSS/JS 直接放進 public 資料夾，完整保留原始設計。

這樣的好處是：
1. 不會跑版，原本長怎樣就怎樣
2. 維護簡單，改原本的 code 就好
3. 未來要加新工具，只要在入口頁加一張卡片就行
